### 十六进制颜色正则

```js
/^#?([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$/.test(str);
```

### QQ 号码正则

```js
/^[1-9][0-9]{4,10}$/.test(str);
```

### 匹配 html 标签

```js
/<(.*)>(.*)<\/(.*)>|<(.*)\/>/.test(str);
```

### 微博话题 匹配`#话题#`

```js
/#[^#]+#/;
```

### 微博匹配`@用户名`

```js
/@[\u4e00-\u9fa5a-zA-Z0-9_-]{4,30}/;
```

### 护照

规则： 14/15 开头 + 7 位数字, G + 8 位数字, P + 7 位数字, S/D + 7 或 8 位数字,等

样本： 141234567, G12345678, P1234567

```js
/^([a-zA-z]|[0-9]){5,17}$/;
```
