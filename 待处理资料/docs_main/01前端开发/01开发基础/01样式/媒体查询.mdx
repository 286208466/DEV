判断设备横竖屏

```css
/* 横屏 */
@media all and (orientation: landscape) {
}
/* 竖屏 */
@media all and (orientation: portrait) {
}
```

判断设备宽⾼

```css
/* 设备宽度⼤于 320px ⼩于 640px */
@media all and (min-width: 320px) and (max-width: 640px) {
}
```
