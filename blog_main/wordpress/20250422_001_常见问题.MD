### 文件上传限制

修改php.ini

```bash
vi /etc/php.ini


post_max_size=512M
upload_max_filesize=512M
max_execution_time=3000
max_input_vars=10000
memory_limit = 512M


extension=curl
allow_url_fopen = On
disable_functions = 

#; 而不是这样 ↓
#; disable_functions = fsockopen,pfsockopen,curl_exec

```

重启php-fpm
```
systemctl restart php-fpm
```
