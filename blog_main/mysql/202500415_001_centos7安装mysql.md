首先检查 是否有 MySQL****正在运行

```bash
ps ajx | grep mysql          // 检查 是否有 mysql 的进程

ps ajx | grep mariabd        // 检查 是否有 mariabd 的进程
```

如果发现 有进程在 运行 需要关闭 进程

```bash
systemctl stop mysqld // 关闭进程
```

检查 是否有 MySQL**** 的安装包

```bash
rpm -qa | grep mysql    // 检查是否有安装包
```

若有安装包出现，并且我们之前没有用过 MySQL ,那就将这些安装包删除。

```bash
rpm -qa | grep mysql | xargs yum -y remove
```

检查 是否有 MySQL** 的残留和配置文件**

```bash
ls /etc/my.cnf    // 检查是否有 配置文件
```

```bash
rm -rf /etc/my.cnf    // 删除配置文件
```

```bash
which mysql         // ------------ 检查 是否有客户端

which mysqld        // ------------ 检查 是否有服务端
```

在 mysql 官网上找到对应的版本

http://repo.mysql.com/

将下载 好的 安装包 上传到 Linxu 上

解压 刚才的安装包

```bash
rpm -ivh mysql57-community-release-el7.rpm
```

检查是否解压成功

```bash
ls /etc/yum.repos.d/ -l    // 检查是否解压成功
```

### 开始安装 MySQL

```bash
yum install -y mysql-community-server
```

可能会出现 GPG 密钥过期

解决方案：输入一下命令，更新 GPG 密钥

```bash
rpm --import https://repo.mysql.com/RPM-GPG-KEY-mysql-2022
```

之后，重新开始安装

**检查是否安装成功**

```bash
which mysql         // ------------ 检查 是否有客户端

which mysqld        // ------------ 检查 是否有服务端
```

### 启动 MySQL

```bash
systemctl start mysqld
```

检查是否成功启动

```bash
ps ajx | grep mysqld    // 检查进程是否启动成功
```

### 登录 MySQL

获取临时密码

```bash
grep "temporary password" /var/log/mysqld.log
```

登入到 mysql

```bash
mysql -u root -p
```

修改密码强度

```bash
SHOW VARIABLES LIKE 'validate_password%';

set global validate_password_policy=0;
set global validate_password_policy=LOW;
set global validate_password_length=4;
```

```bash
mysql> set PASSWORD = PASSWORD('wordpressadmin');
```




查看数据库

```bash
show databases;    //-------------- 就可以看见我们的数据库啦
```

### 给 MySQL 做一些简单的小配置

**为了 防止后期 ，MySQL 出现编码问题 ，我们需要对编码，进行配置**

```bash
vim /etc/my.cnf   // 进入MySQL 的配置文件

port=3306
character-set-server=utf8
default-storage-engine=innodb
```

**配置远程连接**

修改 MySQL 配置文件

```bash
vi /etc/my.cnf
```



编辑 MySQL 的配置文件 /etc/my.cnf，在 [mysqld] 部分添加或修改以下行：

```bash
bind-address = 0.0.0.0
```

保存并退出文件后，重启MySQL服务以使更改生效：

```
sudo systemctl restart mysqld
```

登录 mysql：

```bash
mysql -u root -h localhost -p

use mysql; #打开mysql数据库
```

将 host 设置为%表示任何 ip 都能连接 mysql，当然也可以将 host 指定为某个 ip

```bash

GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'password';

flush privileges; 
```

**关闭远程连接**

```bash
use mysql #打开mysql数据库

#将host设置为localhost表示只能本地连接mysql

update user set host='localhost' where user='root';

#刷新权限表，使配置生效

flush privileges;
```

**不使用 root，专门创建一个用户来用于远程访问**

```bash
CREATE USER 'username'@'%' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON *.* TO 'username'@'%' WITH GRANT OPTION;
FLUSH PRIVILEGES;
```
